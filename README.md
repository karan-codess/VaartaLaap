# 🤖 VaartaLaap

**VaartaLaap** is a plug-and-play AI chatbot widget that can be embedded into any website with minimal setup. Built using the **MERN Stack**, it enables businesses, portfolios, startups, and service websites to provide intelligent conversational support through an easy-to-integrate widget.

---

## 🚀 Features

* 💬 Embeddable chatbot widget for any website
* 🧠 AI-powered conversational experience
* 🔍 Embedding-based knowledge retrieval (RAG)
* 📚 Website-specific knowledge base support
* 👤 Visitor management and conversation tracking
* 📝 Chat history storage
* ⚡ Fast and responsive UI
* 🎨 Bootstrap-powered modern design
* 🔗 Easy integration with external websites
* 📱 Mobile-friendly and responsive
